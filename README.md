# 🏫 슬기로운 학교 생활

중고등학생들에게 "바이브코딩으로 이런 것도 만들 수 있다"는 걸 보여주기 위해 **혼자 예시로 제작한** 학교생활 대시보드 웹사이트입니다. 이후 학생들이 직접 따라 만들어보는 수업/실습용 샘플로 활용할 목적으로 제작했습니다.

**🔗 사이트 바로가기:** [https://atomtutor.github.io/schoollife/](https://atomtutor.github.io/schoollife/)

---

## 📌 프로젝트 소개

"매일 접속하고 싶어지는 학교생활 웹사이트"를 콘셉트로, ChatGPT·Claude·Gemini 등 AI 코딩 도구를 활용해 프롬프트만으로 완성한 예시 프로젝트입니다. 학생들에게 바이브코딩 실습을 안내하기 전, 결과물이 어느 수준까지 나올 수 있는지 보여주기 위한 데모 사이트입니다.

- **제작 시기**: 2026년 7월
- **제작자**: 1인 제작 (학생 실습용 예시 자료)
- **제작 원칙**: 전 과정 무료 진행 — 서버, DB, 유료 API 없이 순수 프론트엔드로만 동작

## 🛠 사용 기술

| 분류 | 기술 |
|---|---|
| 프론트엔드 | HTML, CSS, JavaScript (프레임워크 없음) |
| AI 코딩 도구 | ChatGPT, Claude, Gemini |
| 챗봇 | [Dify](https://dify.ai) (지식 검색 + LLM 기반 영어 단어 챗봇) |
| 배포 | GitHub Pages |
| 디자인 | 글래스모피즘(Glassmorphism), 반응형 레이아웃 |

## ✨ 주요 기능

### 🔮 메인 화면
- **오늘의 운세**: 오늘 날짜 기준으로 매일 같은 결과가 나오는 운세 카드 (문구 + 행운의 색 + 행운의 아이템)
- **카운트다운 타이머**: 점심/하교 시각을 직접 입력해서 남은 시간을 실시간으로 확인

### 🎮 머리 식히기
- **🦖 공룡 달리기**: 스페이스바로 장애물을 피하는 Canvas 게임
- **🍉 수박 게임**: 같은 과일을 합쳐 더 큰 과일을 만드는 격자 기반 퍼즐 게임
- **🧱 벽돌 깨기**: 마우스로 패들을 조작해 벽돌을 깨는 게임

### 📚 공부하자
- **⏱️ 뽀모도로 타이머**: 5/10/30/60분 집중 타이머
- **🎧 백색소음 플레이어**: Web Audio API로 실시간 생성하는 백색소음·비·선풍기·숲 소리 (별도 음원 파일 불필요)
- **📇 영단어 플래시카드**: 원하는 범위·개수만큼 카드를 만들어 학습, 외운 단어/다시 볼 단어 체크
- **📝 실전 단어 학습**: 주관식으로 단어 뜻을 입력해보는 퀴즈

### 🤖 AI 영어 튜터 (전 페이지 공통)
- 우측 하단 플로팅 버튼으로 언제든 호출
- 수능연계 1800단어 지식 문서 기반 Dify 챗봇 연동
- 영어 단어 질문 → 한글 뜻 안내 / 한글 질문 → 영어 단어 + 예문 안내
- 목록에 없는 단어는 일반 지식으로 답변하되 "목록에 없다"는 점을 안내

## 📁 폴더 구조

```
schoollife/
├── index.html                 # 메인 화면 (오늘의 운세, 카운트다운)
├── css/
│   └── style.css               # 전체 공통 디자인, CSS 변수
├── js/
│   ├── main.js                 # GNB 활성화, 타이머, AI 튜터 패널 공통 로직
│   ├── config.js                # 학교별 설정값 (하교/점심 시간 등)
│   └── fortune.js               # 오늘의 운세 로직
├── assets/
│   ├── icons/                  # 파비콘 등
│   ├── images/                 # 게임 이미지
│   └── sounds/                 # 게임 효과음 (CC0 라이선스)
└── pages/
    ├── game.html / study.html         # 게임・학습 허브
    ├── game_dino.html                  # 공룡 달리기
    ├── game_suika.html                 # 수박 게임
    ├── game_brick.html                 # 벽돌 깨기
    ├── pomodoro.html                   # 뽀모도로 타이머
    ├── whitenoise.html                 # 백색소음 플레이어
    ├── flashcard.html                  # 영단어 플래시카드
    └── quiz.html                       # 실전 단어 학습
```

## 🔒 개인정보 및 보안

- 로그인, 회원가입, 개인정보 수집 기능 없음
- 필요한 개인화(오늘의 운세, 타이머 설정 등)는 브라우저 `localStorage`에만 저장
- 게임 이미지·효과음은 CC0 등 무료 라이선스 소스만 사용

## ⚠️ 알려진 제한사항

- **Dify 무료 크레딧**: AI 영어 튜터는 Dify 무료(Sandbox) 플랜을 사용해요. 사용량이 많아지면 응답이 지연되거나 안내 메시지가 뜰 수 있습니다.
- **AI 튜터 새 대화 시작**: 브라우저 보안 정책상 대화 내역을 저희 쪽에서 자동으로 초기화할 수 없어요. 새 단어로 다시 시작하려면 채팅창 안의 🔄 버튼을 눌러주세요.

## 👥 제작

교사(제작자) 1인 제작 · 학생 대상 바이브코딩 실습 예시 자료 · 피드백 및 문의는 사이트 하단 링크를 이용해주세요.

## 📎 학생 실습 활용 안내

이 사이트는 학생들이 그대로 베끼는 것이 목적이 아니라, "AI에게 프롬프트로 이 정도까지 요청할 수 있다"는 걸 직접 보여주기 위한 예시입니다. 학생들과 실습할 때는:

- 이 사이트의 폴더 구조·디자인 콘셉트를 참고 템플릿으로 제시
- 학생들이 직접 자신만의 메뉴 구성(예: 급식 대신 다른 콘텐츠, 다른 게임 등)으로 바꿔보도록 안내
- 프롬프트 작성 → 코드 생성 → 오류 확인 → 재질문의 반복 과정 자체를 실습 포인트로 강조

하면 좋습니다.
